## Push Changes

1. Stage all changes:
```bash
git -C /Users/talk_to_hand/.gemini/antigravity/scratch/automation-base add .
```

2. Commit with descriptive message:
```bash
git -C /Users/talk_to_hand/.gemini/antigravity/scratch/automation-base commit -m "your commit message"
```

3. Push to GitHub:
```bash
git -C /Users/talk_to_hand/.gemini/antigravity/scratch/automation-base push origin main
```
